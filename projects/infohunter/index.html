InfoHunter

可运行
领域： 信息监控

AI 驱动的信息监控与情报分析系统，支持多源抓取、异步分析、趋势报告、飞书推送和 MCP 集成。

技术栈

Python FastAPI MySQL SQLAlchemy APScheduler httpx Docker

核心亮点

  • 抓取、存储、AI 分析、摘要推送分层解耦
  • 支持多内容源订阅与趋势报告生成
  • 集成 Feishu、MCP 和 Web Dashboard

价值创造

痛点

技术趋势、社媒内容和行业信号分散在多个平台，人工跟踪容易遗漏关键变化。

方案

构建多源采集、异步分析、结构化摘要和主动推送链路，让信息监控变成可配置的后台系统。

成果

把信息获取从人工刷新转为自动订阅和摘要消费，适合作为个人研发情报中枢。
